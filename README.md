Данные были получены следующим образом:
```
Первая API - api.spacexdata.com

```

В первом варианте данные предоставлены только до конца 2022 года, а кроме того, начиная с ноября 2022 данные были неадекватно записаны разработчиком API.
Используя второй вариант, были получены данные с ноября 2022 до декабря 2025. Затем обе CSV были объединены и согласованы названия стартовых площадок.
